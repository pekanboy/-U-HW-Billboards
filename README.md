### Что сделать перед запуском?

1) Скачать Node LTS и nvm(<https://github.com/nvm-sh/nvm>)
2) nvm use
3) npm i

А потом запускаем сервер и клиент:
* Запуск сервера - npm run server-start
* Запуск клиента - npm start
